##工具
1. git 常用命令
  git status 查看未提交文件状态
  git add . 添加当前文件目录所有文件到暂存中
  git commit -m "xxxx" 提交并备注
  git commit --amend --no-edit 追加之前的文件（相当于编辑修改之前的提交）
  git push 推送新代码
